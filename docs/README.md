# Documentación del Proyecto

Esta carpeta contiene la documentación técnica y funcional del proyecto TFC MIDTECH – Asistente Compliance IA.

## Contenido

### Documento_Diseño.md

Documento principal de diseño del proyecto.

Incluye:

- Objetivos.
- Arquitectura.
- Tecnologías utilizadas.
- Justificación de decisiones técnicas.
- Funcionamiento del sistema.
- Organización del repositorio.
- Descripción de los workflows.

## Tecnologías principales

- OpenAI GPT-4o
- OpenAI Embeddings
- n8n
- GitHub
- Retrieval-Augmented Generation (RAG)

## Objetivo

Diseñar e implementar un asistente basado en Inteligencia Artificial capaz de consultar documentación corporativa y normativa para apoyar tareas de compliance, ciberseguridad y auditoría.
