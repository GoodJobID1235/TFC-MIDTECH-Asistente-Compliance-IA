# Prompts

Esta carpeta contiene los prompts utilizados para documentar las principales capacidades funcionales del Asistente Compliance IA MIDTECH.

## Capacidades documentadas

### Capacidad 1 - Análisis de Políticas

Evaluación de políticas de seguridad utilizando documentación corporativa y normativa recuperada mediante RAG.

### Capacidad 2 - Gap Analysis

Identificación de diferencias entre la documentación corporativa y los requisitos normativos aplicables.

### Capacidad 3 - Auditoría

Análisis comparativo entre controles documentados y requisitos de cumplimiento.

### Capacidad 4 - Propuestas de Mejora

Generación de recomendaciones priorizadas basadas en las carencias detectadas durante el análisis.

## Tecnologías utilizadas

- OpenAI GPT-4o
- OpenAI Embeddings
- n8n
- GitHub
- Retrieval-Augmented Generation (RAG)

## Objetivo

Documentar las capacidades funcionales implementadas por el asistente para apoyar tareas de compliance, auditoría y mejora continua de la seguridad de la información.
