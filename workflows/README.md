# Workflows

Esta carpeta contiene los flujos de trabajo desarrollados en n8n para el proyecto TFC MIDTECH – Asistente Compliance IA.

## Contenido

### Workflow_Principal_MIDTECH_Asistente_Compliance_IA.json

Workflow principal del proyecto que implementa la arquitectura Retrieval-Augmented Generation (RAG) utilizada por el asistente de compliance.

### explicacion_flujo.md

Documento con la descripción técnica detallada del funcionamiento del workflow y de los componentes que forman la solución.

## Tecnologías utilizadas

- n8n
- OpenAI GPT-4o
- OpenAI Embeddings
- GitHub
- Retrieval-Augmented Generation (RAG)

## Funcionalidades principales

- Consulta documental inteligente.
- Recuperación semántica de información.
- Análisis de cumplimiento normativo.
- Evaluación de políticas de seguridad.
- Detección de deficiencias documentales.
- Generación asistida de informes de auditoría.

## Objetivo

Permitir consultas especializadas sobre documentación corporativa y normativa mediante Inteligencia Artificial Generativa y técnicas de recuperación documental semántica.
